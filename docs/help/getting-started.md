# Getting Started with Thunder Text

Welcome to Thunder Text! This guide will help you install the app and generate your first AI-powered product description.

## Installation

### Step 1: Install from Shopify App Store

1. Visit the [Thunder Text listing](https://apps.shopify.com/thunder-text) in the Shopify App Store
2. Click **"Add app"**
3. Review the permissions Thunder Text requests
4. Click **"Install app"** to authorize

### Step 2: Grant Permissions

Thunder Text needs the following permissions to function:

- ✅ **Read products** - View your product catalog
- ✅ **Write products** - Update product descriptions
- ✅ **Read store information** - Access your shop settings

These are standard Shopify app permissions and are required for Thunder Text to generate and update product descriptions.

### Step 3: Welcome Screen

After installation, you'll see the Thunder Text dashboard with:
- Quick start guide
- Sample product descriptions
- Usage statistics (starts at 0)
- Link to help documentation

## Your First Product Description

### Option 1: Generate from Products Page

1. Navigate to **Products** in your Thunder Text dashboard
2. Click on any product without a description
3. Click the **"Generate Description"** button
4. Wait 5-10 seconds for AI generation
5. Review the generated description
6. Click **"Save to Shopify"** to publish

### Option 2: Bulk Generation

1. Go to **Products** page
2. Select multiple products using checkboxes
3. Click **"Generate Descriptions"** in the bulk actions menu
4. Wait for generation to complete
5. Review each description
6. Approve or edit before publishing

### 🤖 AI-Powered Generation

Thunder Text uses OpenAI's advanced language models to create:
- Compelling product titles (optional)

- Persuasive calls-to-action

The AI analyzes your product images to:

After generation, you can:

Track your usage with:

## Best Practices

### For Best Results

1. **Use High-Quality Images**
   - Clear, well-lit product photos
   - Multiple angles when possible
   - Close-ups of important features

2. **Provide Context**
   - Add product category/type
   - Include basic specifications
   - Note target audience

3. **Review Before Publishing**
   - Always check generated content
   - Verify technical accuracy
   - Ensure brand voice consistency

4. **Iterate and Improve**
   - Regenerate if not satisfied
   - Edit to match your brand
   - Learn what works for your products

### Tips for Quality Descriptions

- ✅ Use multiple product images
- ✅ Fill out product fields in Shopify (category, tags, etc.)
- ✅ Review and edit AI suggestions
- ✅ Test different generation settings
- ❌ Don't publish without review
- ❌ Don't use AI for products with complex technical specs without verification
