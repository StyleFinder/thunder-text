# Coach Store Assignments

## How to Apply

Copy and paste the SQL from `supabase/migrations/update_coach_assignments.sql` into the Supabase SQL Editor:

1. Go to: https://supabase.com/dashboard/project/nrtarnyqxnnfgrfxtvhv/sql/new
2. Paste the SQL
3. Click "Run"

## After Migration

Once you run the SQL:

1. Refresh the BHB Dashboard (http://localhost:3050/admin/bhb-dashboard)
2. Each store will show a blue coach badge
3. Select a coach from the "Filter by Coach" dropdown to see their 2 assigned stores
4. Coaches can star additional stores beyond their assignments
